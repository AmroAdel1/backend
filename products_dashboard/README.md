# Products Dashboard

## Overview
Simple Products Dashboard for Adding, Editing and Deleting Products.

## Features
- Table of Products List
- Add Product
- Edit Product
- Delete Product

## Database
- MySQL
- PHPMyAdmin
- XAMPP

## How to Run
- Import the database in PHPMyAdmin
- Run server (Eg: Apache) from XAMPP
- Go to http://localhost/products_dashboard/
